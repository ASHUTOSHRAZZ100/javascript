## Data Type Conversion

- **Converting in Number**

| values | typeof output | output|
|--------|---------------|-------|
| 33 | number | 33 |
| "33" | number | 33 |
| "635abab" | number | NaN|
| "ashu" | number |  NaN|
| null | number | 0|
|undefine | number | NaN|
| true | number | 1|
|false | number | 0|
|NaN | number | NaN |
